<hr>
<ol type="1">

    <li>Na criação adicionar um arquivo <b>READ ME</b></li>
    <li>Ir nos settings do repositório</li>
    <li>Abra a aba <b>pages</b></li>
    <li>Selecione a branch onde seu arquivo <b>READ ME</b> esta</li>

</ol>
